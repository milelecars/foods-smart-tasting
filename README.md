# Smart Digital Tasting Portal

A full-stack web application for digitalizing snack-tasting sessions with admin analytics and participant feedback system.

## Features

- **Admin Dashboard**: Manage tasting rounds, view analytics, export data
- **Participant Flow**: Email-based session start, sequential tasting, rating system
- **Analytics**: Charts, statistics, and insights
- **Responsive Design**: Works on desktop, tablet, and mobile

## Tech Stack

- Backend: Laravel 9+
- Frontend: Blade templates with Tailwind CSS
- Database: MySQL
- Charts: Chart.js
